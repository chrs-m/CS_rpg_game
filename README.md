# CS RPG Game

Made by [Christopher Micheal](https://github.com/chrs-m) & [Theo Sandell](https://github.com/theo0165).

### Features

- Mini text based RPG game made with C#.
- Made in Swedish and with some easter eggs from Gothenburg.
- 10 rounds.
- Food and weapons.
- Shop and money.
- Emenies and boss.
